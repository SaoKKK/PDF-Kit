Search in the Document

● Perform Search in PDF Document　

To search the words in the document, show navigator pane and input the character string that you want to search into the upper search field there. (Click the menu item "Edit" → "Find" → "Find in PDF" (shift + command + F), then navigator pane will be shown and search field will be highlighted.)
Press the return key and determine input after inputting, then application starts to search the words. The match is performed without regard to the case of alphabetic characters.

● View and Use the Search Result　

The search results are displayed in a list in navigator pane. The left column of the list indicates the pages that contains the finded words. The right column of the list indicates the finded words and the character strings before and after the finded words. And the total number of search results will be indicated in the header of the right column. Click the each row of list, then application scrolls view to show the finded words and highlights them.
