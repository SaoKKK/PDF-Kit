文書のセキュリティ

● パスワードによる文書の暗号化　

　Sao's PDF KitはPDFドキュメントにパスワードを設定し、暗号化することで、ユーザの権限を制限することができます。PDFドキュメントには以下の2つのパスワードの設定が可能です。

  • オープン・パスワード（ユーザ・パスワード）
    • このパスワードでセキュリティ・ロックされたドキュメントは、ドキュメントを開く際にパスワードを請求します。
  • オーナー・パスワード
    • このパスワードを使用すると、ユーザの編集／コピー／プリント操作に関する権限を制限することができます。権限の設定の変更にはオーナー・パスワードが必要になります。
    　ドキュメントが両方のパスワードで保護されている場合は、どちらかのパスワードを入力することでドキュメントを開くことができます。ただし、権限の変更ができるのは、オーナー・パスワードだけです。

    ● ドキュメントのロック解除　

    オープン・パスワードが設定されたPDFドキュメントを開くには、ロックされたドキュメントを開く際に表示されるパスワードを要求するテキストフィールドにオープン・パスワード（ユーザ・パスワード）あるいはオーナー・パスワードを入力してください（上図参照）。

    ドキュメントの完全な権限を有効にするには、「ファイル」メニューから「パスワードを入力...」を選択して、表示されたダイアログにオーナー・パスワードを入力してください。また、ユーザが許可されていない操作を試みた際にも、アプリケーションは類似のダイアログを表示し、パスワードを請求します。その際に、オーナー・パスワードを入力することでも、完全な権限を有効にすることが可能です。

    ● パスワードの設定　

    To set password to the document and encrypt it, select the menu item "PDF" → "Encrypt Document". Then the dialog sheet appears, as shown in the following figure.

    当ヘルプの "パスワードによる文書の暗号化" を参考にして、必要なパスワードをパスワード・フィールドとその確認用フィールドに入力してください。ユーザの編集／コピー／プリント操作に対して権限ロックを掛けたい場合は2つのチェックボックスを使用します。「ドキュメントの内容のコピーを禁止する」にチェックを入れた場合、ユーザはドキュメントをコピーすることができません。「ドキュメントの印刷を禁止する」にチェックを入れた場合、ユーザはドキュメントをプリントすることができません。どちらか一方でもチェックが入っていれば、ユーザはドキュメントを編集／保存することができません。
    パスワードには最大32文字までのローマ字が設定できます。大文字と小文字は区別されます。 必要なセッティングを終えて、「暗号化」ボタンをクリックしてください。暗号化が実行されます。.
    暗号化の実行後にファイルメニューから保存／別名で保存を行った場合、暗号化は解除されてしまうので注意してください。

    - 注意事項 -
    パスワードを忘れた場合、ドキュメントを元の状態に復元することはできません。パスワードで保護する前にファイルのバックアップコピーを保存することを推奨します。

    ● 暗号化の解除　

    暗号化を解除するには、ドキュメントをオーナー・パスワードで開いた後、「ファイル」メニューから「保存...」か「別名で保存...」を選択してください。保存後のファイルは暗号が解除されています。
