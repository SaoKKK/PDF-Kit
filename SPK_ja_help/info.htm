文書情報の表示／編集

● 文書情報の表示　

文書情報パネルを表示するとPDF文書のさまざまな情報を取得することができます。文書情報パネルを表示するには、「PDF」メニューから「文書情報を表示」（shift + command + I）を選択してください。
以下が表示される情報の内容です。

一般情報
  • ファイル名 : ドキュメントのファイル名
  • ファイル・パス : ドキュメントのファイル・パス
  • 作成日 : ドキュメントの作成日時
  • 更新日 : ドキュメントを最後に保存した日時
    • 詳細情報
  • バージョン : ドキュメントのバージョン
  • ページ : ドキュメントのページ総数
  • セキュリティ : ドキュメントが暗号化されているかどうか
  • コピー : ドキュメントのコンテンツをクリップボードにコピーする権限の有無
  • プリント : ドキュメントのプリント権限の有無
  • コンテンツの作成 : ドキュメントの内容を作成したアプリケーション名
  • PDFデータの作成 : PDFデータを作成したアプリケーション名
    • ドキュメントの概説
  • タイトル : ドキュメントのタイトル
  • 著者 : ドキュメントの著者
  • 件名 : ドキュメントの件名の説明
  • キーワード : ドキュメントのキーワード

    • ● 文書情報の編集　

    ドキュメントの概説は編集することができます。編集するには、左下の[ ]ボタンをクリックしてください。すると、ドキュメントの概説の値を格納しているテキストフィールドが編集可能になります。値を編集して「更新」ボタンをクリックしてください。キーワードにはカンマ区切りで複数の文字を登録することが可能です。

    ✳︎ ドキュメントがパスワードで保護されていて、編集権限がロックされている場合、[ ]ボタンをクリックすると、パスワードを請求するダイアログシートが表示されます。ドキュメントの概説を編集するには、オーナー・パスワードを入力してください。
