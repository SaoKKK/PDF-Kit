Managing Security

● About Encrypting with Password　

Sao's PDF Kit can encrypt PDF document with passwords and restrict user's permissions. PDF document can have the following two passwords.

  • Open password (User password)
    • The document encrypted with this password requires the password when it's opened.
  • Owner password
    • With this password, you can limit the user's authority for editing/copying/printing operations. To change the limiting requires owner password.
    If the document is protected with two password, it can be open with a password of either. But only owner password can change the authority.

    ● Unlock the Document　

    To open the document which is setted open password (user password), input the user or owner password to the text field that is displayed when you try to open it (see upper figure).

    If you need to enable full permissions of the document, select the menu item "File" → "Input Password..." and input the owner password to the dialog's password field. And when you try operation that you don't have permission, the application will display that requires owner password. You can enable full permissions by inputting the owner password to its password field, too.
