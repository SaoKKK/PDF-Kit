Managing Security

● About Encrypting with Password　

Sao's PDF Kit can encrypt PDF document with passwords and restrict user's permissions. PDF document can have the following two passwords.

  • Open password (User password)
    • The document encrypted with this password requires the password when it's opened.
  • Owner password
    • With this password, you can limit the user's authority for editing/copying/printing operations. To change the limiting requires owner password.
    If the document is protected with two password, it can be open with a password of either. But only owner password can change the authority.

    ● Unlock the Document　

    To open the document which is setted open password (user password), input the user or owner password to the text field that is displayed when you try to open it (see upper figure).

    If you need to enable full permissions of the document, select the menu item "File" → "Input Password..." and input the owner password to the dialog's password field. And when you try operation that you don't have permission, the application will display that requires owner password. You can enable full permissions by inputting the owner password to its password field, too.

    ● Set Password　

    To set password to the document and encrypt it, select the menu item "PDF" → "Encrypt Document". Then the dialog sheet appears, as shown in the following figure.

    Use "About Encrypting with Password" as a reference and input passwords you need. When you need to limit the user's authority for editing/copying/printing operations, use two check boxes. When you switch on the check box "Forbid Copying Contents", users cannot copy contents of the document. When you switch on the check box "Forbid Printing Document", users cannot print the document. When either check box is on, users cannot edit and save the document.
    You can set a maximum of 32 roman characters as password. It is case sensitive.
    After setting up, click the button "Encrypt".
    Note that if you use the menu item "Save..." or "Save as..." after encrypting, the document will be decrypted.

    - ATTENTION -
    If you lose password, the document cannot be restored to the original state. Please store backup copy of the PDF before protecting it by password.

    ● Decrypt the Document　

    If you need to decrypt the document, open the document with owner password and use the menu item "Save..." or "Save as...".
