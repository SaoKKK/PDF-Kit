Merge PDF

● Merge PDF　

1. Structure of the window "Merge PDF"

To merge PDF files, click the menu item "PDF" → "Merge PDF..." to show the window "Merge PDF" first.
The files that will be merged will be registered to the left list and you can confirm the preview of a registered file in the right pane. The buttons under the list are used to edit the list. You can perform merging with the right-most button at the bottom of the window after registering the files.

2. Register the Files that will be Merged

If the window "Merge PDF" is displayed for the first time after the application was launched, the opened documents will be added to the list of the window automatically. If there's no opened document, the list will be empty. To add files to the list, drag and drop PDF files into the list from "Finder" or click "+" button under the list and choose PDF files with the file dialog and click the button "Open".
When you need to remove files from the list, choose files that you want to remove on the list and click "-" button under the list or press command + delete key. If you want to clear all files, click "Clear" button under the list.
The files will be merged according to order of lists. When you want to change the order of the files, move the items by drag and drop on the list.

3. Confirmation of the each Files and Registration of Pages that will be Merged

When a file is selected on the list, the preview of the file will be viewed in the right pane. The way to navigate between the pages is similar to document window. Use the buttons at the bottom of the pane or the menu items of "Go".
After registering files, set the page range for every each file. Choose "All Pages" from the combo box, if you want to merge all pages from the file. If pages should be specified, choose the second row of the combo box and input pages in Comma Separated Value following the example format.

4. perform merging

After the registrations are all done, click the right-most button "Merge PDF" at the bottom of the window. If the check box "Create Bookmark" is on, the application will add bookmarks to the created file automatically. The destination of the bookmark is the top page of the each source files and the label of the bookmark is the file name of the each source files. If the state of the chack box is off, the created file has no bookmarks.
When merging processing is finished, the created file is displayed as a new document. Specify the file path and save the document.

● Temporarily Store Processing　

When the button "Store Work State" is clicked, the application stores the state of registration on the list to the temporary file. This is kept even after application is finished. Use it in the cases that registration is not settled with once. Click the button "Restore Work State" to restore the state of the registration to the list from temporary file. Please note it is only the state of the registration that the application saves. PDF files themselves are not stored by this processing.
