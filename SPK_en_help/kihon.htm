Basic Operations

● Working with PDF Files　

Sao's PDF Kit can open/close existing PDF files, and can save them with the same name or a different name after editing, and can return the document to saved status last. Those procedures are as follows.

1. Open/Close PDF Files

To open PDF documents, select the menu item "File" → "Open" (command + O) and select PDF files in the Open dialog sheet and click the button "Open", or drag and drop PDF files to the application icon. If you are setting Sao's PDF Kit as the default application to open PDF files, double click will open PDF files.
To know about opening document which is locked with password, see "Managing Security". To close a PDF file, select the menu item "File" → "Close" (command + W) or click the close button of the document window. When you press option key to select the menu item "Close", the application will close all opened windows.

2. Save PDF Files

When you select the menu item "File" → "Save..." (command + S), the application saves current document to the same file path. When you save the PDF file that was created with Sao's PDF Kit, select the menu item "File" → "Save..." and set the file path in Save dialog and click the button "Save". If you need to save the current document to different file path, select the menu item "File" → "Save As..." (shift + command + S) and set the file path in Save dialog and click the button "Save".

3. Return the Document to Saved Status Last

If you need to abandon all edits and return the document to saved status last after editing it, select the menu item "File" → "Revert to Saved". Please note that all of the current edits will be lost.

4. Create new PDF document from clipboard item

Sao's PDF Kit can create a new PDF document from a clipboard item when clipboard contains the data of image type. Select the menu item "File" → "New Document from Clipboard" (command + N) to perform.

● View PDF　

【Figure 1: Structure of Document Window】　

1. Display Mode

【Figure 2: Buttons to Switch Display Mode】

You can use four following display mode when viewing PDF documents:

  • Single Page
    • Display one page at a time
  • Single Page Continuous
    • Display one page in a horizontal direction and display all pages continuously in a vertical direction
  • Two Pages
    • Display two facing-pages at a time
  • Two Pages Continuous
    • Display two facing-pages in a horizontal direction and display all pages continuously in a vertical direction
  • Display as Book
    • When the current display mode is "Two Pages" or "Two Pages Continuous", treat the document's first page as a book cover.

    To switch display mode, use the menu item "View" or the right-most buttons on the bottom of the document window. Select the menu item corresponding to the display mode which you expect from "Single Page" (command + 1), "Single Page Continuous" (command + 2), "Two Pages" (command + 3) and "Two Pages Continuous" (command + 4) of the "View" menu. When the buttons to switch display mode are clicked, the application changes the display mode to corresponding mode to the clicked button, too. See figure 2 about which display mode is corresponding to each button. To swich display mode to "Display as Book", select the menu item "Display as Book" (command + 5).

    2. Paging through a Document

    【Figure 3: Go Buttons Group】

    To go to another page, use the menu item "Go" or "Go Buttons Group" (left-most buttons group of the bottom of document window). To go to the next page, select the menu item "Next Page" (command + ↓) or click the button []. To go to the previous page, select the menu item "Previous Page" (command + ↑) or click the button []. To go to the first page, select the menu item "First Page" or click the button []. To go to the last page, select the menu item "Last Page" or click the button [].
    The text field of the left of "/" indicates the current page and the one of the right indicates the total pages of the document. Input the page number which you want to see and press Enter key, then the specified page will be displayed.
    By using the menu items "Go Back" and "Go Forward" or the button [] and [], the application stores a history of the movement of the page. To step back on the page history, select the menu item "Go Back" or click the button []. To step forward on the page history, select the menu item "Go Forward" or click the button [].
    As other way for paging through a document, the application provides "Scroll Tool ()". While "Scroll Tool" of "Tool Button Group" is selected, you can control scrolling with mouse movement. Select "Scroll Tool", and move the mouse on the document. Then press left-mouse button and drag it toward the direction opposite to the direction you want to see. The document will be moved through the distance of the mouse movement.

    3. Change the Magnification

    【Figure 4: Zoom Buttons Group】

    To change the magnification, use the menu item "View" or "Zoom Button Group" or "Zoom Tool".
    To zoom in, select the menu item "Zoom In" (shift + command + >) or click the button [] or select "Zoom Tool" from "Tool Button Group" and click on the document.
    To zoom out, select the menu item "Zoom Out" (shift + command + <) or click the button [] or select "Zoom Tool" from "Tool Button Group" and option-click on the document. To view the document with its actual size, select the menu item "Acrual Size" (command + 0).
    To resize the document to fit document pane, select the menu item "Zoom to Fit" (command + 9) or click the button []. When the display mode "Single Page" or "Two Pages" is selected, the application changes the magnification of the document to the maximum size that the whole page fits into the document pane. When the display mode "Single Page Continuous" or "Two Pages Continuous" is selected, the application changes the magnification of the document to the size that the width of the page fits a document pane.
    When you want to zoom in a specified area, select "Zoom Tool", and drag to draw a rectangle around the area to magnify. Then the application resizes document to fit the area to the document pane and displays the area in the center of document pane.
    When view reaches the limit of the magnification / reduction, the button and the menu item which reached the limit is disabled. If "Zoom Tool" is selected, the cursor changes to [] to notice reaching the limit.

    4. Full Screen View

    Document windows and the window "Merge PDF" which appears when the menu item "Merge PDF" is clicked can be seen in full screen view. To see them in full screen view, click the menu item "View" → "Enter Full Screen" (control + command + F). And when you want the window to exit full screen view, click the menu item "View" → "Exit Full Screen" (control + command + F).

    ● Print Document　

    Click menu item [File] → [Page Setup...] (shift + command + P) to set general printing options. Theavailableoptions will vary with different printers and drivers. See your printer driver documentation for details.
    Click menu item [File] → [Print...] (command + P) to print document. Specify the printer, page range, number of copies, and other options, and click "Print" button.

    ● Navigator Pane　

    【Figure 5: Navigator Pane】

    The "Navigator Pane" is an area located on the left side of the document window (In figure 1, it is displayed in blue color). Objects which navigates in the document are displayed there. You can display a thumbnail view or an outline view there, and when you searth any word in the document, the search result will be displayed there.(About searching in the document, see "Search in the Document").
    To show or hide "Navigator Pane", click the left upper button of the document window.
    Thumbnail view arranges thumbnail images of each page of the PDF document in a line in order of pages. To display thumbnail view in navigator pane, click the left button of "Switch Navigator Button".
    An outline is the information such as the headline of the PDF document having hierarchical structure. The PDF document can have a marker called "bookmark" which has the label and the positional information in the document in any place. The structure which arranges them hierarchically is an outline. To display outline view in navigator pane, click the right button of "Switch Navigator Button". If the document has no bookmark, the application tell you that with display dialog, and navigator pane won't change.
    About create/edit bookmarks, see "Working with Bookmark".

    ● Create Selection　

    【Figure 6: Create Selection】

    Sao's PDF Kit provides two functions to create selection and the created selection will be used by other editing processing. They can be switched by switching the button "Letter Selection Tool" and the button "Area Selection Tool".

    "Letter Selection Tool" should be used as follows;

  • To select any letters, mouse down to the first letter and drag to the last letter.
  • To select a word, double click upon the word.
  • To select a line, triple click upon the line.
  • To select the letters in any area, hold down option key as you drag across the area which contains letters you want to select.
  • To select all of the text in the document, select the menu item "Edit" → "Select All" (command + A)

    • When "Area Selection Tool" is selected, drag acress the area that you want to select. You can move or resize the area even after it was created. The selection which is created with "Area Selection Tool" is limited in arbitrary one page. content across a plurality of pages. When "Area Selection Tool" is selected, the menu item "Edit" → "Select All" will create the selected area which covers whole of the current page.

    Regardless of the selected tool, to deselect, click anywhere outside the selected objects or select the menu item "Edit" → "Deselect" (shift + command + A)
