Sao's PDF Kit helps you view and edit PDF documents. It is compatible with Mac OS 10.9 or later.

Sao's PDF Kit is an application which has generic function to view, navigate, print PDF documents. It provides the other ability to edit PDF documents as below.

  • Search words in a PDF document
  • Merge plural PDF files to one file
  • Split a PDF file constructed in plural pages into some files
  • Remove pages, change the order of pages
  • Make or edit bookmarks
  • Extract text from PDF documents
  • Export pages as image files
