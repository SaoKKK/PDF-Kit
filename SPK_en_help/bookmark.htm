Working with Bookmark

● Use Bookmark　

The PDF document can have a marker called "bookmark" which has the label and the positional information in the document in any place. The structure which arranges them hierarchically is called outline.
If document has an outline, Sao's PDF Kit provids the means to display and use it. You can create new bookmarks and edit them with this application, too.
Bookmarks navigate in the PDF document according to their position information. When you click a bookmark, document pane displays its destination. And if the outline view is in "Preview Mode", the selected row of the outline is changed in sync with the movement of the page. About the mode of outline view, see the next chapter.

● Mode of Outline View　

【Buttons for switching outline view mode】

The outline view has two modes.
Under "Preview Mode", the outline is specialized in ability to guide, and the selected row of the outline changes by the linkage with movement of the page. You cannot edit the label of the bookmark. It is the default mode.
"Edit Mode" is the mode for editing. Under this mode, the selected row of the outline doesn't synchronize with movement of the page, and you can edit the label of the bookmark on the outline view.
To switch the mode, click the lower button of the outline view (See an upper figure). When a new bookmark is created, the application changes it to "Edit Mode" automatically.

● Bookmark Panel　

【Structure of Bookmark Panel】

Bookmark panel is used to display the information of a bookmark, create/edit it. To show/hide it, select the menu item "Window" → "Show/Hide Bookmark Panel" (shift + command + B). When you create a new bookmark, the application shows it automatically.
When a bookmark is selected on outline view, its information will be indicated there. The value of "Label" is the title of the bookmark, it is same as the one indicated in the outline view. The value of "Index" of "Destination" indicates the zero-based serial number of the page. The value of "Label" of "Destination" indicates the title of the page. It is 1-based consecutive numbers generally, but it can be a specific character string. "Point" of "Destination" is the position in the document that is displayed when you click the bookmark. The application moves a document so that this coordinate is located on the upper left of the document pane. The origin of the coordinate is the lower-left corner of the page. And the numerical value increases towards top-right corner.　

● Create New Bookmark　

To create a new bookmark, use the menu item "Bookmark" or the button "New Bookmark from Data" of Bookmark panel. Each item works as follows.

  • the menu item "New Bookmark"
    • A new bookmark whose label is "Untitled" and whose destination is top left corner of page whose index is 0 is created. Change the data of each item later as needed. For editing them, See the next chapter.
  • the menu item "New Bookmark from Selection"
    • A new bookmark whose label is the character strings contained in selection and whose destination is top left corner of the selection is created. Select this menu item after making selection.
  • the button "New Bookmark from Data" of Bookmark panel
    • A new bookmark is created from the data of panel.
    No matter which making method is selected, when a bookmark is selected, a new bookmark will be added at the end of the child items of the selected bookmark, and when nothing is selected on outline view a new bookmark will be added at the end of the outline. You can change a hierarchy or the order of the added bookmark later. For changing it, See "Change Hierarchy and Order".

    ● Edit Bookmark　

    You can perform the change of the label of the bookmark from the outline view or the bookmark panel. To change from the outline view, switch the mode of outline view to "Edit Mode" first (When a new bookmark is created, it changes to "Edit Mode" automatically). Under "Edit Mode", the field for the label is editable. Select the row which you want to change the label, and push enter key, and input the new label.
    To change from the bookmark panel, select the row which you want to change the label. And input the new label to label field of panel and click the button "Update Selected Bookmark Data".
    To change the destination of a bookmark, select the row which you want to change the destination. And input new destination on bookmark panel and click the button "Update Selected Bookmark Data".

    ● Change Hierarchy and Order　

    To change a hierarchy or the order of outlines, move a bookmark by drag & drop on the outline view.
