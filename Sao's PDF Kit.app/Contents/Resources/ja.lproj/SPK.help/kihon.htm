基本操作

● ファイルの基本操作　

Sao's PDF Kitは既存のPDFファイルを開くあるいは閉じる、編集後のPDFファイルを保存あるいは別名で保存する、編集中のPDFファイルを最後に保存した時の状態に戻すことができ、それらの手順は以下のようになります。

1. ファイルを開く／閉じる

　Sao's PDF KitでPDFファイルを開くにはファイルメニューの「開く...」（command + O）を選択して、表示されるファイル選択パネルで開きたいPDFファイルを選択するか、ファインダーでアプリケーション・アイコンに向かって開きたいPDFファイルをドラッグアンドドロップします。また、PDFファイルを開くデフォルト・アプリケーションとしてSao's PDF Kitを設定している場合は開きたいPDFファイルをダブルクリックすることで開くことができます。
　ファイルを閉じるには、ファイルメニューの「閉じる」（command + W）を選択するか、ウインドウのクローズボタンをクリックしてください。オブションキーを押しながら「閉じる」メニューを選択すると、開いているすべてのウインドウが閉じられます。

2. ファイルを保存する

　ファイルメニューの「保存」（command + S）を選択すると、ファイルを上書き保存します。Sao's PDF Kitで作成したPDFファイルを保存する時は、ファイル名と保存場所を選択するためのダイアログが表示されますので、それぞれの項目を設定して「保存」ボタンをクリックしてください。開いているファイルを別の名前で保存、別の場所に保存したい場合は、ファイルメニューの「別名で保存...」（shift + command + S）を選択し、表示されたダイアログにて、ファイル名と保存場所を設定して「保存」ボタンをクリックしてください。

3. ドキュメントの復元

　ドキュメントに何らかの編集を施した後に、編集結果を破棄して、最後に保存した時の状態に復元する場合は、「ファイル」メニューから「最後の保存に戻す」を選択してください。復元後に編集後の状態に戻すことはできないことにご留意ください。

4. クリップボードから新規ドキュメントを作成する

　クリップボードにイメージタイプのデータが含まれる時、Sao's PDF Kitはそのデータから新規ドキュメントを作成することができます。実行には、「ファイル」メニューから「クリップボードから新規ドキュメント作成」（command + N）を選択してください。

● 閲覧の基本操作　

【図1：メイン・ウインドウ各部の名称】　

1. 表示モードの切り替え

【図2：表示モード切り替えボタン】

　ドキュメントの表示は「単ページ表示」「単ページ連続表示」「見開表示」「見開連続表示」の4種類のモードに切り替えることができます。「単ページ表示」は1度に1ページを表示します。「単ページ連続表示」は横方向には1ページ表示し、縦方向には連続してすべてのページを表示します。「見開表示」は横方向に見開で2ページを表示します。「見開連続表示」は横方向に見開で2ページ表示し、縦方向には連続してすべてのページを表示します。切り替えは「表示」メニューか、ドキュメント・ウインドウ右下の「表示モード切り替えボタン」から行います。「表示」メニューの「単ページ表示」（command + 1）、「単ページ連続表示」（command + 2）、「見開表示」（command + 3）、「見開連続表示」（command + 4）を選択することで、それぞれ対応する表示モードに切り替えられます。「表示モード切り替えボタン」はそれぞれのボタンに対応した表示モードに切り替えます。それぞれのボタンにどの表示モードが対応するかについては図2をご覧ください。

2. ページの移動

【図3：移動ボタン・グループ】

　ページの移動は「移動」メニューか、ドキュメント・ウインドウ左下の「移動ボタングループ」から行います。「移動」メニューのメニュー項目「前ページへ」（command + ↑）、「次ページへ」（command + ↓）、「最初のページへ」、「最後のページへ」あるいは「移動ボタングループ」の「前ページ」、「次ページ」、「最初のページ」、「最後のページ」はそれぞれの名前が示すページへ移動します。また、「移動ボタングループ」に属するテキスト・フィールドは"/"の左側が現在のページ、右側が総ページ数を示しています。「現在のページ」テキスト・フィールドに、移動したいページを入力してEnterキーで確定させると、入力したページに移動します。また、「移動」メニューのメニュー項目「戻る」、「進む」あるいは「移動ボタングループ」の「戻る」「進む」を使用すると、移動したページの履歴が残り、「戻る」で履歴を1つ戻る、「進む」で履歴を1つ進むことができるようになります。
　このほか、「ツール・ボタン・グループ」のスクロールツール（左から3番目のボタン）を選択している間は、マウスの動きによって移動を制御することができます。スクロールツール選択後、文書の上にマウスダウンし、進みたい方向と逆方向にドラッグしてください。そうすると、ドラッグした分の距離だけ移動します。

3. 表示倍率の変更

【図4：ズームボタン・グループ】

　表示倍率の変更は「表示」メニューか、「ズームボタン・グループ」から行います。現在の表示より一段階拡大して文書を表示するには、「表示」メニューの「拡大」（shift + command + >）を選択するか、「ズームボタン・グループ」の「拡大」ボタンをクリックするか、「ツール・ボタン・グループ」のズームツール（一番右のボタン）を選択した状態で文書の上をクリックします。現在の表示より一段階縮小して文書を表示するには、「表示」メニューの「縮小」（shift + command + <）を選択するか、「ズームボタン・グループ」の「縮小」ボタンをクリックするか、「ツール・ボタン・グループ」のズームツールを選択した状態で、optionキーを押しながら文書の上をクリックします。文書を実際の大きさで表示するには、「表示」メニューから「実際のサイズ」（command + 0）を選択します。「表示」メニューの「ウインドウに合わせる」（command + 9）を選択するか、「ズームボタン・グループ」の「ウインドウに合わせる」ボタンをクリックすると、現在のウインドウの大きさに合うように文書の表示倍率を変更します。表示モードが「単ページ表示」と「見開表示」の場合、「ウインドウに合わせる」を選択すると、アプリケーションは上下左右がウインドウに収まる最大の大きさに文書を倍率変更します。表示モードが「単ページ連続表示」と「見開連続表示」の場合には、アプリケーションは文書の横幅がウインドウの大きさにフィットするように、文書を倍率変更します。
　また、ズームツールを選択した状態で、文書内の拡大したい領域の端の位置でマウスダウンし、そのままドラッグすると、その領域を囲むように矩形が点線で表示されます。拡大したい領域の逆端の位置までマウスをドラッグしてから離すと、点線で囲まれていた領域がウインドウの文書表示枠にフィットするように拡大表示されます。表示が拡大／縮小の限界に達すると、限界に達したボタン・メニューは無効化されます。ズームツールを選択中の場合は、カーソルの形状が"符号なしルーペ ()"に変化し、拡大あるいは縮小の限界に達したことを知らせます。

4. フルスクリーンによる閲覧

　ドキュメント・ウインドウと「PDF」メニューの「PDFを結合...」選択時に表示されるPDFを結合ウインドウは、フルスクリーンによる閲覧が可能です。フルスクリーンにするには、「表示」メニューの「フルスクリーンにする」（control + command + F）を選択してください。フルスクリーンを解除するには、「表示」メニューの「フルスクリーンを解除」（control + command + F）を選択してください。

● 文書の印刷　

　用紙サイズ等、一般的な印刷オプションを設定するには、「ファイル」メニューから「印刷書式...」（shift + command + P）を選択します。使用できるオプションは、プリンタやプリンタドライバによって異なります。詳しくは、プリンタドライバのマニュアルを参照してください。
　PDF文書を印刷するには、「ファイル」メニューの「印刷...」（command + P）を選択します。プリンタ、印刷範囲、印刷部数、および他のオプションを指定して、「プリント」をクリックします。

● ナビゲータ・ペイン　

【図5：ナビゲータ・ペイン】

　ナビゲータ・ペインとは、メイン・ウインドウの左側にある文書内をナビゲートする役割を持つオブジェクトが表示される領域のことを指します（図1に青で示された領域です）。必要に応じ、文書のサムネールやアウトラインなどを切り替えて表示することができるほか、文書内の検索を実行した時には、検索の結果がここに表示されます（文書内の検索について、詳しくは文書内の検索をご覧ください）。
　ナビゲータ・ペインの表示／非表示を切り替えるには、メイン・ウインドウ左上の「ナビゲータ・ペイン表示切り替えボタン」をクリックします。ナビゲータ・ペインが表示されている時はナビゲータ・ペインを隠し、表示されていない時は表示します。
　サムネールとは、文書のすべてページの小さなプレビューを連続的に並べたもののことをいいます。ナビゲータ・ペインにサムネールを表示するには、「ナビゲータ切り替えボタン」の左側をクリックします。
　アウトラインとは、階層構造を持つ文書の見出しなどの情報です。PDF文書は任意の場所に「しおり」と呼ばれるラベルと文書内での位置情報を持つマーカーを設置でき、それらを階層的に並べたものがアウトラインです。ナビゲータ・ペインにアウトラインを表示するには、「ナビゲータ切り替えボタン」の右側をクリックします。文書にしおりが1つも設定されていない場合は、その旨ダイアログで表示され、ナビゲータ・ペインは変更されません。しおりの設定については、しおりの利用と作成／編集をご覧ください。

● 選択範囲の作成　

【図6：選択範囲の作成】

　Sao's PDF Kitでは文書内の任意の範囲を選択するために、2つの方法が用意されており、作成された選択範囲をほかの編集作業に利用することができます。選択方法は「ツール・ボタン・グループ」の「文字選択ツール」と「エリア選択ツール」の切り替えによって切り替えることができます。

　「文字選択ツール」の使用方法は以下の通りです。

  • 任意の文字を選択するには、選択したい文字の最初の文字にマウスダウンして最後の文字までドラッグします。
  • 単語を選択するには、選択したい単語の上でダブルクリックします。
  • 行を選択するには、選択したい行の上でトリプルクリックします。
  • 任意のエリア内の文字を選択するには、オプションキーを押しながら選択したい文字を含むエリアをドラッグして囲みます。
  • 文書内のすべての文字を選択するには、メニュー「編集」から「すべて選択」（command + A）を選択します。

    • 　「エリア選択ツール」選択時には、選択したい領域の端の位置にマウスダウンし、そのままマウスドラッグすることで選択領域を文書上に描画し始め、選択したい領域の逆端でマウスを離すと選択領域が完成されます。作成された選択エリアは、作成後も、自由に移動・変形ができます。「エリア選択ツール」で作成できる選択範囲はページ内に限られます。複数のページにまたがって選択範囲を作成することはできません。「エリア選択ツール」選択時に、メニュー「編集」から「すべて選択」を選択した場合は、現在のページ全体を覆う選択範囲が作成されます。
    　選択を解除したい時は、「編集」メニューから「選択を解除」（shift + command + A）を選択するか、選択ツールが選択された状態で選択領域以外をシングルクリックしてください。
