Indicate/Edit Document Info

● Indicate the Document Info　

You can get various information of the PDF document on the information panel. To display the information panel, select the menu item "PDF" → "Show Document Info" (shift + command + I).
The following is the indicated information.

General Information
  • File Name : File name of the document
  • File Path : File path of the document
  • Creation Date : the document’s creation date
  • Modification Date : the document’s last-modified date
    • Detailed Information
  • Version : Version of the document
  • Page : Total page number of the document
  • Security : Value specifying whether the document is encrypted (Encrypted or None)
  • Copy : The permission to copy of document's content to the clipboard
  • Print : The permission to print the document
  • Creator : The name of the application that created the document content
  • Producer : The name of the application that produced the PDF data for the document
    • Document Description
  • Title : Title of the document
  • Author : Author of the document
  • Subject : a description of the subject of the document
  • Keyword : keywords for the document

    • ● Edit the Document Info　

    The document descriptions are editable. To edit them, click the button [ ]. Then the value fields will be editable. Edit them and click the button "Update". You can register plural words with a keyword in Comma Separated Value.

    ✳︎ If the document is encrypted with password and you don't have permission to edit it, when you click the button [ ], the application displays the dialog sheet to require the owner password. To edit the desctiptions, input the owner password to it.
